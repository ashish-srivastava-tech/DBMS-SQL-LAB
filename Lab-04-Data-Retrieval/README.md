# Lab 04 – Data Retrieval Using SQL

## Objective
To practice retrieving data from relational database tables using SELECT queries in Oracle SQL.

## Description
This lab focuses on executing read-only SQL queries to retrieve data from existing tables.  
It includes usage of:
- Column selection with aliases
- Filtering using WHERE clause
- Sorting using ORDER BY
- Limiting rows using ROWNUM
- Derived/calculated columns

## Files Included
- Lab_04_Solution.sql – Contains all SELECT queries
- Lab_04_Questions.pdf – Assignment questions provided by faculty

## Tools Used
- Oracle Database
- Oracle SQL Developer / SQL*Plus
- GitHub

## Learning Outcomes
- Understanding SELECT statements
- Applying filtering conditions
- Sorting and limiting results
- Creating derived attributes
- Working with Oracle-specific syntax
