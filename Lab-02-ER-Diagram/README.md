# Lab 02 – ER Model & Relational Schema

## Objective
To design an ER diagram for a university database and convert it into
a relational schema using Oracle SQL.

## Contents
- ER-Diagram.png : ER diagram designed using draw.io
- lab2_tables.sql : Oracle SQL CREATE TABLE statements
- README.md : Description of the lab

## Tools Used
- draw.io 
- Oracle Database
- Oracle SQL Developer / SQL*Plus

## Notes
The Student–Course many-to-many relationship is resolved using the
Enrollment table with a composite primary key.
