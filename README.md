# 🎯 Repository Overview

This repository contains structured **Database Management Systems (DBMS)** laboratory work performed using **Oracle Database**.

All SQL programs are:

- Written in standard SQL (Oracle compatible)
- Executed using Oracle SQL Developer / SQL*Plus
- Structured lab-wise for clarity and maintainability
- Designed following relational database principles

---

# 🗂 Repository Structure

The repository is organized lab-wise.

Each lab folder typically contains:

- 📄 SQL solution script (`.sql`)
- 📄 Lab question PDF
- 📄 Lab-specific `README.md`
- 📊 CSV data files (if required)
- 📐 ER Diagram / Schema files (where applicable)
